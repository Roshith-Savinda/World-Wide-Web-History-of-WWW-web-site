BROWSER SOFTWARE

Browser software allows a user to access and interact with websites (written in HTML, translated into readable content) on the internet. Most browsers support external plugins required to display active content, e.g. in-page video, audio, and Flash content. Browsers are available with different features and are designed to run on different operating systems.

To qualify for inclusion in the Browser category, a product must:

  1. Be downloadable
  2. Translate HTML into readable content
  3. Support external plugins to display active content

Key Benefits of Browser Software

  1. load websites using URL domains
  2. View and share content from around the web with your personal and professional networks
  3. Tap into the limitless information stored across the web for research or task-related purposes
  4. Perform important job duties by accessing web-based applications
